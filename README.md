## Функціонал програми

Консольний додаток дозволяє зручно керувати розкладом занять. Основні можливості:
1. **Додавання занять:** введення дня тижня, часу та назви предмета з автоматичною перевіркою та форматуванням даних.
2. **Перегляд розкладу:** виведення всього збереженого розкладу в зручному текстовому форматі.
3. **Пошук за предметом:** пошук занять за ключовим словом (без урахування регістру).
4. **Збереження даних:** усі записи автоматично зберігаються у файл `data/schedule.txt`, тому розклад не втрачається після закриття програми.
